Introduction

Welcome to Trivial Compute! This user's manual will guide you through setting up and playing the game. Trivial Compute is an educational version of Trivial Pursuit, designed to make learning fun and interactive.

Managing Categories and Questions

Before starting the game, you can manage categories and questions to customize the game content. You can add, edit, and delete categories and questions as needed.

Categories are the subjects or topics for the questions in the game. You can create new categories, and edit and delete categories. You can manage categories using the "Manage Categories" table. All changes are saved in real-time.

Questions are the content of the game. You can create new questions, assign them to categories, and edit or delete questions. You can manage questions using the "Manage Questions" table. All changes are saved in real-time.

Gameplay

Trivial Compute is a turn-based game where players roll dice and answer questions to advance on the game board. The game board is a grid composed of 45 squares, representing 4 categories with 10 questions each, as well as four "roll again" squares, and one "home square" where all players start and finish the game.

  1. The current player rolls the dice
  2. The player moves their token
  3. The player answers the question that is presented to them. If they landed on a "roll again" square, they may roll again instead.
  4. If the player answers correctly, they may roll again. If they were on a HQ square, then they gain a point for that category.
  5. If the player answers incorrectly, the turn passes to the next player.
  6. The first player to reach the home square with a point in each category wins the game.
